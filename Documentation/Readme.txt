1- Lancer le serveur (fichier server.js) avec nodejs en console
2- Lancer l'application dans un navigateur via l'adresse suivante : 
	127.0.0.1:8087/index.html
(On utilise le port 87 en localhost, et on l'ance l'application via le fichier index.html)

3- Utilisation : 
	- M : Mode création et déplacement
	- S : Mode changement de taille
	- C : Mode conditions limites
	- E : Mode exportation pour envoyer les données au serveur et afficher le maillage des géométries
